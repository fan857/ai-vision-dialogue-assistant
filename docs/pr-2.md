# PR2：摄像头实时预览功能

## 功能概述

实现摄像头实时预览功能，用户可以在前端页面打开摄像头并看到实时画面。

## 实现细节

1. 添加摄像头预览区域
2. 实现打开/关闭摄像头功能
3. 显示摄像头状态（未开启/请求中/已开启/失败）
4. 处理权限请求和错误情况
5. 更新功能列表状态

## 技术实现

### 前端（Vue3）

- 使用 `navigator.mediaDevices.getUserMedia` API 访问摄像头
- 支持视频流的播放和停止
- 添加摄像头控制按钮和状态显示
- 响应式设计，支持移动端

### 状态管理

- `isCameraActive`：摄像头是否开启
- `cameraStatus`：摄像头状态（idle/requesting/active/failed）
- `mediaStream`：存储媒体流对象

### 样式设计

- 摄像头预览区域占满宽度，最大640px
- 灰色背景占位符，显示状态提示
- 渐变按钮，悬停效果
- 响应式布局，支持移动端

## 测试方式

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:5173

### 功能测试

1. 点击"打开摄像头"按钮
2. 浏览器弹出权限请求，点击允许
3. 页面显示实时摄像头画面
4. 点击"关闭摄像头"按钮
5. 摄像头停止，页面显示占位符

### 错误测试

1. 拒绝摄像头权限
2. 无摄像头设备
3. 摄像头被其他应用占用

## 限制

- 只在前端实现，不需要后端支持
- 不涉及图片上传或截图功能
- 不支持音频录制
- 不实现AI模型调用
